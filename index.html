This game is built on bleeding edge web development technology. There is some setup involved if you're on an older sytem:
  • THE ESSENTIAL FACTOR IS ENABLING WEBGL. Currently this is found in newer versions of Chrome and Mozilla, but you may still have to enable it. You can do this by typing "chrome://flags" (without the quotes in your web browser URL. Don't get intimidated. Simply search for the selection that allows you to enable WebGl. You may also need to enable the experimental system's worker option, if WebGL isn't enough.
  • Currently working on switching Multiplayer from running off of WEBRTC workers. Instead have it run along side a NODEJS server with integration into Facebooks API system so you can shoot your friends! It's coming. I'll figure it out.

What...you made a video game?

Nope. Can't take all the credit. WebGL use case has been around for a couple years now. However, we have finally reahed a point where Browsers are natively integrating more and more with GPU's (your graphics cards). Graphics cards are more powerful than they have ever been, and cheaper than they have ever been as well. It's the perfect storm for WebGL to move to the forefront in Web Development. This completely transforms dynamic design and development. While still in its infancy, I am more than confident over the next year this kind of tech is going to explode.

This game is unique in that its completely in browser. Not impressed? You're right, Java has been doing this for a long time. Python can create these environments as well. However, the big bugaboo with the other guys it 1.) Security: 90% of the malware you have on your inferior PC right now is from accidently downloading Java droplets. Way to easy to hack. 2.) These kind of games generally require a Client and a Server to play. An example would be your xbox, which acts as a Client, connects to a video games server in order to function. I didn't want to make you guys download anything. Refer back to point numero uno. This is solely written in Javascript, HTML, and CSS. The EXACT same components you find in Facebook, Tumblr, Youtube, JoesBlogs, EVERYWHERE. I find it absolutley remorkable. This version is still buggy, but I will continue to work on it. As the knowledge base grows on this technology, so will the game.
